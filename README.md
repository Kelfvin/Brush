# Brush

## 简介

一款命令行环境下的刷题软件，功能：

* 支持用户自行导入json题库

* 目前软件自带的题库有<u>军事理论</u>和<u>近代史</u>

* 有错题收集和复习的功能

![21.12.28](21.12.28.gif)


## 关于json导入

```json
{
    "name":"科目名称",
    "sections":{
        "章节名称(如：第0章)":{
            "problems":{
                "题目一(如:1. (单选题) 我是不是最帅的？>_<)":{
                    "key":"题目的答案（如：A）",
                    "option":"题目的选项如（A. 是的\nB. 是的\nC. 是的\nD. 是的）",
                    "type":"题的种类（如：选择题）"
                },
                "题目二":""

            }
        },
        "章节2xx":"xxx"
    }
}
```
